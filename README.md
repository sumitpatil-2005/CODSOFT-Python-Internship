# CODSOFT Python Internship Tasks

This repository contains the tasks completed for the CODSOFT Python Programming Internship.

## Task 1 – To-Do List
A command-line application to manage tasks.
Features:
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks

## Task 2 – Calculator 
A simple calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division
The program runs continuously until the user selects Exit.

## Task 3 – Password Generator
Generates strong and meaningful passwords using random characters.

Language Used: Python
